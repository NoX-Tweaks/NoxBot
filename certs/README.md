# Certificados privados

Coloque aqui os arquivos exigidos pelo MongoDB apenas no ambiente de hospedagem:

- `ca-certificate.crt`
- `certificate.pem`
- `private-key.key`

Esses arquivos sao sensiveis e ficam ignorados pelo Git. Na SquareCloud, envie-os como arquivos privados do app ou inclua no pacote de deploy sem commitar em repositorio publico.
